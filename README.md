# Galaksen – Romutforsker

Interaktiv romapp med Aurora-design som kan installeres på mobilen din (PWA).

## Funksjoner
- 🪐 **Planetskanner** – animert solsystem med informasjon om alle 8 planeter og solen
- ⭐ **Stjerner** – detaljert info om 8 kjente stjerner i universet
- 📱 **Installer på mobil** – fungerer som en ekte app (Progressive Web App)
- 🌐 **Offline-støtte** – fungerer uten internett etter første besøk

## Installer på mobil
1. Åpne appen i nettleseren din (Chrome / Safari)
2. Trykk på **"Installer"**-banneret nederst på skjermen
3. Appen legges til på hjemskjermen din

> **iOS**: Trykk på Del-ikonet → «Legg til på Hjem-skjerm»
